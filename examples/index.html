switch_access Examples

Give AAC/switch users switch/keyboard navigation on your webpage with highlighing and other neat functions

If you have a webpage where you want to provide accessability with switch control. Simply add the coffeescript or compiled javascript to your webpage and add "switch-element" or "switch-element-1", "switch-element-2" etc. For only using "switch-element" the option for sort_element_list_after_numbers must be set to false.
